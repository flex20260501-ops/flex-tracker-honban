💡 給与計算式

1. 社数効率を算出：社数効率 = 回収数 ÷ 稼働時間
2. 効率に応じた基準時給：最低¥1,400 〜 効率0.1上がるごとに+¥100
3. 職級加算：G1 +¥200 / G2 +¥100 / G3 +¥0
合計時給 = 基準時給 + 職級加算　　月給 = 合計時給 × 稼働時間

☁️ クラウド同期（Firebase）

リアルタイム同期を有効にすると、責任者が更新したデータが瞬時に全メンバーへ反映されます。
Firebase Console で無料プロジェクトを作成 → 「ウェブアプリを追加」→ 表示される設定情報を以下に入力してください。

📖 設定手順を見る
  1. console.firebase.google.com にGoogleアカウントでログイン
  2. 「プロジェクトを追加」→ プロジェクト名（例：flex-tracker）を入力
  3. Google アナリティクスは任意（無効でOK）
  4. 左メニュー「Firestore Database」→「データベースを作成」
    ※「本番モード」を選択 → ロケーション asia-northeast1 (東京)
  5. セキュリティルールを以下に貼り付け：
    rules_version = '2';
    service cloud.firestore {
      match /databases/{db}/documents {
        match /{document=**} {
          allow read, write: if true;
        }
      }
    }     ※ チーム内利用前提のため、認証なしのオープンルール
  6. プロジェクト概要 → 歯車アイコン → 「プロジェクトの設定」
  7. 「マイアプリ」セクションで「ウェブ </>」アイコンをクリック → アプリ名を入力 → 「アプリを登録」
  8. 表示される firebaseConfig = {...} の内容を以下に転記
⚠️ 共有方法： 設定保存後、このHTMLファイルをそのままコピーしてメンバーに配布してください。設定情報がファイル内に保存されているため、開いた瞬間に同期が始まります。
🔒 セキュリティ： 設定情報はチーム外に流出しないようご注意ください（流出すると第三者がデータ閲覧可能になります）
